# PDWEL-MVC
Projeto MVC de um Sistema de Controle de Hotel
• Arquitetura MVC.  
• Aplicação do Java EE/Jakarta EE.  
• CSS, Javascript.  
• PostgreSQL, Hibernate, jQuery, Bootstrap, Datatables, entre outros.
